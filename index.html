What's OpenSOC?

OpenSOC is a collection of open source tools using biggest data and machine learning algorithms for discovering network anomalies in a large computer network.

Components

OpenSOC consists primarily of two main components: the portal / UI, and the backend machine learning cluster. The two communicate primarily through Apache Kafka.
